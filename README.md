# HomeLab
Growing repository for bootstraping a private HomeLab server with Ansible

Providing apps:
- PiHole
- Heimdall dashboard
- Home Assistant
- Portainer
- Shlink
- Speedtest Tracker
- Traefik
- Wireguard

Future upgrades:
- #100DaysHomeLab challange
- adding openHAB
- automated backups
- converting HomeLab to k3s
